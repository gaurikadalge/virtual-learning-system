# Virtual Learning System

## Overview
A C++ based Object-Oriented Programming project that simulates a virtual learning platform. The project demonstrates core OOP principles such as classes, encapsulation, modular design, and separation of concerns.

## Features
- User and course management
- Modular and extensible architecture
- Clear object-oriented structure

## Tech Stack
- C++
- Standard Template Library (STL)

## Project Structure
- core/ – core application logic
- Data/ – data files
- models/ – class definitions
- main.cpp – entry point

## How to Run
Compile using a standard C++ compiler (e.g., g++) and execute the generated binary.
